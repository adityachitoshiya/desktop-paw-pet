## ✨ Features

- **Transparent & Borderless:** Lives directly on your desktop, floating above your wallpaper.
- **Interactive:** Follows your mouse, reacts to clicks, and even responds to your typing!
- **Autonomous Behaviors:** If left alone, your pet will randomly sleep, sit, walk, run, scan the area, or sneak around.
- **Multiple Themes:** Choose from default procedurally drawn cats or beautifully crafted sprite sheet cats like **Oreo Cat** and **Aero Cat (Brown)**.
- **Click-Through Mode:** Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows) to enable click-through mode, allowing you to click windows behind the cat without moving it.
- **Customizable:** Change the cat's size and walking speed in the settings.
- **Feed Your Pet:** Drop some virtual fish for your cat to eat!

## 💻 Development

If you want to run the project locally or contribute:

```bash
# Clone the repository
git clone https://github.com/adityachitoshiya/desktop-paw-pet.git

# Go into the repository
cd desktop-paw-pet

# Install dependencies
npm install

# Run the app in development mode
npm run dev
```

### Building for Production
```bash
# Build for macOS
npm run build:mac

# Build for Windows
npm run build:win
```
